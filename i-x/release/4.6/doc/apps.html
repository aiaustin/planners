I-X Applications Files

I-X applications are usually provided as a separate distribution. The files should be added after the main I-X system is created. An application's zip file containing application specific parts should be unzipped into the same directory that the main I-X release is unzipped.

The files add one or more specific apps/<app-name> directories, any associated comms/<comms-strategy> directories and possibly other support files in the relevant directories.
