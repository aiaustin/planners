Introduction

The documentation currently available for I-X covers:

  • introductory texts,
  • using I-X-based systems,
  • configuring I-X to create I-X-based systems,
  • reference documentation (APIs), and
  • background reading.
Below, the documentation for each of these areas is described, before a summary of the main documents is given.

I-X Introduction

  • I-X Release Directory Structure.
  • I-X Introduction, a good starting point.
  • An Introductory Paper on I-X (pdf).

Using I-X-Based Systems

  • The I-X Process Panels - User Guide (pdf) is a guide to using I-X applications.
  • The I-X Domain Editor Guide (pdf) describes the editor and how it is used.
  • I-Demo Basic (pdf) describes a basic demo that shows how to use a single process panel.
  • I-Demo Coop (pdf) describes a demo that shows the use of two cooperating process panel.

Configuring I-X to Create I-X-Based Systems

  • The Configurer's Guide (pdf) provides a "how-to" guide to using, configuring, and extending I-X applications. The demos above also support this guide, illustrating the methods.
  • Adding I-X support for OWL-S describes how to use I-X with the OWL-S API.
  • Adding an I-X application distributed separately describes what to do to download and install additional I-X applications.
  • The reference documentation particularly useful for configuring applications is
    • The I-X Parameters List describes the parameters that can be used with different I-X components.
    • A note about test menus describes how to put together test menus for process panels.
    • I-X Syntax describes the XML syntax for representing objects that I-X systems understand.

Reference Documentation (APIs)

  • I-X parameters list
  • I-X XML Syntax
  • A Set of Notes intended for developers and technical users.
  • Javadoc-generated class documentation.

Background Reading

  • Background reading is available in the form of papers and texts.

Software Licences

  • I-X Licence
  • Software Components With Other Licence Terms

Summary

In summary, the main I-X documnents are:
  • The I-X Process Panels - User Guide is a guide to using I-X applications.
  • The I-X Domain Editor Guide describes the editor and how it is used.
  • The I-X Configurer's Guide provides a "how-to" guide to using, configuring, and extending I-X applications. To support this guide, there is a suite of demonstrators for illustration.
  • A Set of Notes intended for developers and technical users.
  • Javadoc-generated class documentation.
